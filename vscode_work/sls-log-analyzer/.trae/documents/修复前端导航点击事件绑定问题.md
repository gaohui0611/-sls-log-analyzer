## 问题分析

**根本原因**: HTML 和 JavaScript 类名不匹配
- HTML 中导航项使用 `class="nav-item"`
- JavaScript 中使用 `document.querySelectorAll('.tab')` 绑定事件

这导致事件监听器无法正确绑定，点击导航项没有任何响应。

## 修复计划

### 1. 修复 app.js 中的标签页切换事件
将第 66 行的 `.tab` 改为 `.nav-item`：
```javascript
// 修改前
document.querySelectorAll('.tab').forEach(tab => {

// 修改后
document.querySelectorAll('.nav-item').forEach(tab => {
```

### 2. 修复第 70-71 行的类名操作
```javascript
// 修改前
document.querySelectorAll('.tab').forEach(t => t.classList.remove('active'));
document.querySelectorAll('.tab-content').forEach(c => c.classList.remove('active'));

// 修改后
document.querySelectorAll('.nav-item').forEach(t => t.classList.remove('active'));
document.querySelectorAll('.tab-content').forEach(c => c.classList.remove('active'));
```

### 3. 验证修复
- 刷新页面
- 点击各个导航项
- 确认右侧内容正确显示

## 预期效果
修复后，点击左侧导航项，右侧将正确显示对应的页面内容。
